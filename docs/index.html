Installation

1

Clone the Repository

Get the latest version of Pest Patrol from GitHub

git clone <repository-url>
cd pest-patrol
2

Create Virtual Environment

Set up an isolated Python environment (recommended)

python -m venv venv
source venv/bin/activate # On macOS/Linux
venv\Scripts\activate # On Windows
3

Install Dependencies

Install all required packages from requirements.txt

pip install -r requirements.txt
4

Verify Installation

Test that everything is set up correctly

python -c "from src.models import create_model; print('‚úì Installation successful!')"

Configure Training

Edit config.yaml to set your preferences:

name: "resnet50"
pretrained: true

training:
batch_size: 32
num_epochs: 50
3

Train the Model

Start training with a single command:

python train.py
4

Make Predictions

Use the trained model to classify new images:

python predict.py path/to/image.jpg
python predict.py path/to/images/ --top-k 3

Available Models

ResNet Family

ResNet-18, ResNet-34, ResNet-50, ResNet-101, ResNet-152. Excellent balance of accuracy and speed. ResNet-50 recommended for most use cases.

EfficientNet Family

EfficientNet-B0 through B4. Best balance of accuracy and efficiency. Optimized for both performance and resource usage.

MobileNet Family

MobileNet-V3 Small and Large. Optimized for mobile devices and edge deployment. Perfect for real-time applications.
